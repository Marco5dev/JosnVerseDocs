# Drop a data file and reseting it

This methode can clear a total data file to be an empty array 
you can use it with the next example:
```js
db.drop("dataname")
```
yes it's just this simple usage and you can use promisses with it

which means when you wants to reset a data file just use the drop function with the name of the data file
